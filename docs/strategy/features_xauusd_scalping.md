# 特徴量設計：XAUUSD スキャルピング（5分足）

作成日: 2026-04-05

---

## 基本方針

- 対象: XAUUSD（金/ドル）
- 時間足: 5分足ベース（MTFで1分・15分・1時間を補助）
- スタイル: スキャルピング（数分〜数十分での決済を想定）
- スプレッドが広いため、エントリー精度が命

---

## 特徴量一覧

### 【A】価格・ローソク足系

| 特徴量 | 説明 | スキャル向け重要度 |
|---|---|---|
| OHLCV | 基本データ | ★★★ |
| 前足からの変化率 | モメンタムの基礎 | ★★★ |
| ボディ幅 / ヒゲ比率 | ローソク足の形状（勢いの強さ） | ★★★ |
| 直近N本の高値・安値 | サポート・レジスタンス候補 | ★★★ |
| キリ番までの距離 | 2000, 2050, 2100等の心理的節目 | ★★★ |
| 前足終値からのギャップ | 飛びつきエントリーの判断 | ★★ |

**XAUUSDはキリ番（ドルの整数値）で強く反発・反落しやすい。**

---

### 【B】ボラティリティ系（スキャルでは特に重要）

| 特徴量 | 設定 | 説明 |
|---|---|---|
| ATR | 5・14期間 | 値動きの大きさ。TP/SLの基準にも使う |
| ボリンジャーバンド幅 | 20期間・2σ | 収縮（スクイーズ）→ 拡大でエントリー判断 |
| 標準偏差 | 10・20期間 | 短期的な荒れ具合 |
| 高値-安値幅（レンジ） | 直近5・10本 | 今のレンジの広さ |

**スキャルはボラが高い時間帯に絞るのが重要。ATRが小さすぎる時間帯は避ける。**

---

### 【C】モメンタム・トレンド系（短め設定を使う）

スキャルピングでは通常より**短い期間**設定が有効。

| 特徴量 | スキャル向け設定 | 通常設定（参考） |
|---|---|---|
| RSI | **5・9期間** | 14期間 |
| Stochastic | **5,3,3** | 14,3,3 |
| MACD | **5,13,1** | 12,26,9 |
| CCI | **10期間** | 20期間 |
| MA | **5・10・20期間** | 20・75・200 |
| EMA | **5・10期間** | 20・50期間 |
| ROC（変化率） | **3・5期間** | 10期間 |

---

### 【D】XAUUSD固有の特徴量

| 特徴量 | 説明 | 取得方法 |
|---|---|---|
| **DXY（ドル指数）の変化率** | 金と逆相関。DXY上昇→金下落が基本 | MT5でUSDINDEXを取得 |
| **DXYとXAUUSDの乖離** | 相関が崩れたときが逆張りチャンス | 計算で算出 |
| 米10年債利回り方向 | 金利上昇→金下落の傾向 | 別途データ取得（後回しでOK） |

**まずはDXYだけ入れるだけでも効果が出やすい。**

---

### 【E】時間・セッション系

XAUUSDはNYセッションが最重要（FXのロンドン重複より影響大）。

| 特徴量 | 内容 |
|---|---|
| 時刻（sin/cos変換） | 周期性を表現 |
| 曜日 | 金曜NYは荒れやすい |
| **NYセッションフラグ** | 21:00〜翌6:00（JST）★最重要 |
| ロンドンセッションフラグ | 16:00〜25:00（JST） |
| アジアセッションフラグ | 8:00〜17:00（JST）※金は動きにくい |
| **NY指標発表フラグ** | CPI・雇用統計・FOMC前後は別扱い |

**アジア時間はスプレッドが広がりやすく、スキャルには向かない。フィルタリングを推奨。**

---

### 【F】マルチタイムフレーム（補助）

| 時間足 | 使い方 |
|---|---|
| 1分足 | エントリータイミングの微調整 |
| 5分足 | ベース（メインの学習データ） |
| 15分足 | 短期トレンドの方向フィルター |
| 1時間足 | 中期トレンドの方向フィルター |

**「1時間足が上昇トレンド → 5分足で押し目買いのみ」のようなフィルターとして使う。**

---

## 優先度まとめ

```
【まず入れる】
  ATR（5・14）
  RSI（5・9）
  ボリンジャーバンド幅
  キリ番までの距離
  時刻（sin/cos）・NYセッションフラグ
  前足変化率・ボディ幅

【次に追加】
  DXY変化率
  Stochastic（5,3,3）
  MACD（5,13,1）
  1時間足のトレンド方向

【余裕があれば】
  米10年債利回り
  1分足の特徴量
  NY指標発表フラグ
```

## USJPYとの主な違い

| | USDJPY（中期） | XAUUSD（スキャル） |
|---|---|---|
| スプレッド | 小さい | 大きい → エントリー精度が命 |
| 指標の影響 | セッション全体 | NYセッション・米指標に集中 |
| 重要節目 | 丸数値（149・150等） | キリ番（2000・2050等）が強い |
| 相関資産 | 他通貨ペア | DXY・米金利 |
| 適したMA期間 | 20・75・200 | 5・10・20（短め） |
| アジア時間 | 動く | ほぼ動かない（除外推奨） |
