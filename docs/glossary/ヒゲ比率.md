# ヒゲ比率

ローソク足の上ヒゲ・下ヒゲの長さをボディやレンジ全体と比較した比率。

## 読み方
```
上ヒゲが長い = 上値を試したが跳ね返された（売り圧力あり）
下ヒゲが長い = 下値を試したが戻ってきた（買い圧力あり）
```

## コード例
```python
df["upper_wick"] = df["high"] - df[["open","close"]].max(axis=1)
df["lower_wick"] = df[["open","close"]].min(axis=1) - df["low"]
df["upper_wick_ratio"] = df["upper_wick"] / (df["high"] - df["low"])
df["lower_wick_ratio"] = df["lower_wick"] / (df["high"] - df["low"])
```

## 使い方
反転サインの判断に使う。上ヒゲが長いローソクが続く → 上値の重さを示す。
