# NYセッションフラグ

NYセッション（21:00〜翌6:00 JST）中かどうかを 1/0 で表す特徴量。

## コード例
```python
df["is_ny"] = ((df.hour >= 21) | (df.hour < 6)).astype(int)
```

## XAUUSDにおける重要性
XAUUSDはNY時間に最も動くため、「今NYセッションかどうか」がエントリー判断に効く。
アジア時間（8:00〜16:00 JST）はスプレッドが広く動きにくいため除外推奨。

## 3つのセッション（JST）
| セッション | 時間 | XAUUSD への影響 |
|---|---|---|
| 東京 | 8:00〜17:00 | 小（除外推奨） |
| ロンドン | 16:00〜25:00 | 中 |
| NY | 21:00〜翌6:00 | 大（最重要） |
