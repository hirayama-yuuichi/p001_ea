# venv構築とMT5データ取得

## やったこと
- `p001_ea` プロジェクト内に `.venv`（Python 3.14）を作成
- `requirements.txt` を追加（MetaTrader5, pandas, pyarrow）
- `ml/fetch_mt5_data.py` の `DATE_FROM` を MT5 に実際にロードされている最古日（2025-07-21）に修正
- XAUUSD 5分足データを取得・保存

## 結果
- 取得件数: 50,033件（2025-07-21〜2026-04-02）
- 保存先: `data/raw/XAUUSD_M5_20250721_20260405.parquet`（1.5 MB）
- 基本統計: 終値平均 4218.17、最小 3269.21、最大 5587.17、ATR(14) 平均 5.72
