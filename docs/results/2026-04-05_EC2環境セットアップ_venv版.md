# 作業記録: EC2 Windows 環境セットアップを .venv に変更

**日付**: 2026-04-05  
**タイトル**: EC2 Windows セットアップスクリプトを .venv ベースに改版

## やったこと

### 1. requirements-windows.txt を新規作成
- Windows 専用の Python パッケージリストを分離
- MetaTrader5, pandas, numpy, pyarrow, python-dotenv を明示化
- 将来の ML パッケージはコメントで記載

### 2. infra/windows/setup/setup.ps1 を大幅改版
**変更前** (Miniconda + conda):
```powershell
# Miniconda ダウンロード → Miniconda インストール
# → conda env fx59v2 作成 → pip install
```

**変更後** (.venv):
```powershell
# git clone → Python 確認 → .venv 作成 → pip install
```

具体的な改変:
- Miniconda インストール部分を完全削除（5-41行目）
- Python インストール確認機能を追加
  - system Python が PATH に入っているか確認
  - 無い場合は choco install python -y を促す
- .venv 作成（`python -m venv .venv`）
- requirements-windows.txt からのパッケージインストール

### 3. infra/windows/setup/README.md を完全改版
変更点:
- セットアップ流程を .venv に合わせて説明
- 動作確認コマンドを `.venv\Scripts\python.exe` に更新
- トラブルシューティングセクション追加
- ローカル Linux → EC2 SSH 実行の具体例を追加
